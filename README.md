# First to 4 (Connect Four) with AI 🔴🟡

A classic terminal-based "Connect Four" game written in Python. Challenge yourself against three different levels of AI, ranging from a random-move beginner to a strategic mastermind using the Minimax algorithm.

## ✨ Features
* **Two-Player Style:** Play against an AI opponent.
* **Three Difficulty Levels:**
    * **Easy:** Moves randomly.
    * **Medium:** Blocks your immediate wins and takes its own.
    * **Hard:** Uses **Minimax with Alpha-Beta Pruning** to look 5 moves ahead.
* **Colorized Terminal UI:** Uses ANSI colors for better visibility of Red (Player) and Yellow (AI) pieces.
* **Robust Logic:** Handles full columns and invalid inputs gracefully.
